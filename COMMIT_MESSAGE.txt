fix: Corrigir tela branca na Vercel - Instalar Tailwind CSS corretamente

PROBLEMA:
- Tailwind estava sendo carregado via CDN no index.html
- Na Vercel, o CDN demorava ou era bloqueado
- React renderizava sem estilos → tela branca
- Funcionava local (cache), mas quebrava em produção

SOLUÇÃO:
- Instalado Tailwind CSS como dependência (v4.1.18)
- Instalado @tailwindcss/postcss + autoprefixer
- Criado tailwind.config.js e postcss.config.js
- Atualizado index.css com @tailwind directives
- Importado CSS no index.tsx
- Removido CDN do index.html

RESULTADO:
- CSS compilado de 0.51 kB → 9.92 kB (Tailwind incluído)
- App funciona perfeitamente na Vercel
- Build mais rápido (sem esperar CDN)
- Mais seguro (sem scripts externos)
